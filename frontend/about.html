Opis aplikacji – Planety CRUD

Aplikacja Planety CRUD została stworzona w ramach projektu edukacyjnego w języku Python (Flask). Pozwala na zarządzanie danymi o planetach (dodawanie, edycja, usuwanie, przeglądanie).

Autoryzacja

Dostęp do operacji CRUD wymaga zalogowania przy pomocy tokenu JWT. Przykładowe konto testowe: 

Login: testuser
Hasło: 12345

Technologie

  • Python + Flask
  • SQLite
  • HTML + JavaScript (fetch API)
